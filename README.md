# Python编程：从入门到实践（第3版）外星人入侵源码

## 分支说明
- main 分支为原版代码
- develop 分支为改造后的代码

## 目录结构
```
alien_invasion
├── alien.py // 外星人战机
├── alien_invasion.py // 主程序入口
├── bullet.py // 子弹
├── button.py // 按钮
├── font // 字体
├── game_stats.py // 游戏统计信息
├── images // 图片资源
├── README.md
├── scoreboard.py // 得分牌
├── settings.py // 设置
└── ship.py // 玩家战机
```

## 游戏规则说明：
| 按键  | 功能说明  |
| :------------: | :------------: |
| ←  | 战机向左移动  |
| →  | 战机向右移动  |
|  ↑ |  战机向上移动 |
| ↓  | 战机向下移动  |
| 空格(space)  | 发射子弹  |
